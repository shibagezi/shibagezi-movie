## 部署方式

### 方式一：本地预览
1. 直接双击打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python 3
   python3 -m http.server 8000
   
   # 或使用 Node.js
   npx serve .
   ```

### 方式二：部署到服务器
1. 将整个文件夹上传到Web服务器（如Apache、Nginx）
2. 配置域名指向该文件夹
3. 确保服务器支持静态文件服务

### 方式三：使用GitHub Pages
1. 创建GitHub仓库
2. 上传所有文件到仓库
3. 在仓库设置中启用GitHub Pages
4. 访问生成的网址

## 自定义内容

### 修改公司信息
在 `index.html` 中修改以下部分：
- 公司名称：搜索"十八格子影视"进行替换
- 联系方式：在"联系我们"部分修改地址、电话、邮箱
- 团队信息：在"团队介绍"部分修改成员信息

### 修改图片
1. 替换Unsplash图片链接为实际公司图片
2. 建议图片尺寸：
   - 横幅图片：1600×900像素
   - 作品展示：600×400像素
   - 团队成员：400×400像素

### 修改样式
在 `css/style.css` 中：
- 主色调：修改 `#ff6b6b`（红色）为其他颜色
- 字体：修改 `font-family` 设置
- 布局：调整各区块的间距和大小
